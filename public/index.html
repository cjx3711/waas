Words as a service

A modern RESTful API for getting words you need.
v0.1

What?

WAAS (Words as a service) provides a modern, RESTful, scalable solution to the problem of getting words off the internet.

Usage

Content formats

WAAS will respond to the following 'Accept:' values with appropiate content
  • application/json (Default) - Content will be returned as a json object
  • text/plain - Content will be returned as a plain string
  • text/html - Content will be returned as HTML page with a bootstrap hero unit, containing the message and subtitle

Operations

Path Desciption
/word/:text Your gateway to any word known to mankind (currently limited to the ASCII character set)
/strip/:text
/trim/:text		 Removes the whitespace surrounding the text
/upcase/:text
/uppercase/:text		 Converts all text to uppercase
/downcase/:text
/lowercase/:text		 Converts all text to lowercase
/camelcase/:text Converts all text to CamelCase
/snakecase/:text Converts all text to snake_case
